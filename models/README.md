# Models Directory

This directory holds ONNX model files and associated assets used by Deep Cuts'
analysis pipeline. **No model files are committed to git** â€” they are either
generated by the export scripts in `tools/` or downloaded at runtime.

## Generating ONNX models

Set up the Python environment (Python â‰Ą 3.10 required):

```bash
python3 -m venv tools/.venv
tools/.venv/bin/pip install -e tools/
```

Then run the export scripts from the repo root:

```bash
# CLAP audio + text encoders, mel filterbank, tokenizer (~600 MB total)
tools/.venv/bin/python tools/export_clap_onnx.py

# all-MiniLM-L6-v2 sentence encoder + tokenizer (~90 MB)
tools/.venv/bin/python tools/export_sentence_onnx.py
```

For the Essentia classifier mel filterbank (only needed for the future
genre/mood pass), install the `essentia` optional dependency first â€”
`essentia-tensorflow` is available on PyPI including Apple Silicon:

```bash
tools/.venv/bin/pip install -e "tools/[essentia]"
tools/.venv/bin/python tools/export_essentia_mel_weights.py
```

The Discogs-Effnet base model and the genre/mood/vocal head models are
available as ready-to-use ONNX files from the official Essentia model hub â€”
